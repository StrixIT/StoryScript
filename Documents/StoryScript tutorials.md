# StoryScript Tutorials

## Getting started with StoryScript as a beginner

If you have never used StoryScript before, please consult the detailed tutorials on our [website](https://storyscript.nl/tutorials).

## Workflow summary for regular users

If you have already used StoryScript before but need a reminder of the key steps to edit and test your story or game,
you may want to check our [workflow summary](./StoryScript workflow v1.md).

## Learning more about web technologies

To successfully create interactive stories and games with StoryScript, you do not need programming experience, but it is good
to have a good knowledge of HTML and CSS, which you will need to create and style textual and visual content. To learn more, we recommend
the tutorials and hands-on exercises provided by [W3Schools](https://www.w3schools.com). W3Schools is also a great resource
to learn more about [Javascript, "the programming language of the web"](https://www.w3schools.com/js/default.asp).
